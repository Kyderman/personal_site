My Role

Gareth approached me and asked me to create a website for his exciting new art business. The opportunity to create a clean portfolio site was something I wanted to get involved with. The site was a design of my own inspiration, using bootstrap as a base, with various libraries for the gallery and carousels. However, these libraries, particularly the carousel, were modified to fit my design (black and white vs colour elements). The various icon-like images were crafted by myself.

Key Features

  • Single page scrolling with navigation

  • Flickr API integration

  • Responsive Captcha CSS

  • Modern, sleek design
